04 — Setup Steps
Follow the guide
01
Step 1
Create a server
pending ›
Создать сервер, на котором будет работать OpenClaw.
  • Откройте DigitalOcean
  • Нажмите Create → Droplet
  • Выберите Ubuntu 22.04
  • Выберите минимальный тариф

После создания сервер покажет IP-адрес:

Пример IP
134.209.xxx.xxx
🔒 Security note: OpenClaw запускается на отдельном сервере, чтобы изолировать его от вашего компьютера.
02
Step 2
Configure SSH access
pending ›
Настроить безопасный доступ к серверу через SSH-ключ.

Подключение по паролю небезопасно. Создайте SSH-ключ:

bash
ssh-keygen

Добавьте публичный ключ в настройки сервера, затем подключитесь:

bash
ssh root@SERVER_IP
🔒 Security note: SSH-ключи значительно безопаснее паролей и защищают от брутфорса.
03
Step 3
Update the server
pending ›
Обновить систему для устранения известных уязвимостей.
bash
apt update apt upgrade -y
🔒 Security note: Обновления закрывают известные уязвимости системы. Всегда обновляйте сервер перед установкой новых сервисов.
04
Step 4
Install dependencies
pending ›
Установить Git и Docker.
bash
apt install git docker.io -y

Docker используется для запуска OpenClaw в контейнере.

🔒 Security note: Контейнеризация создаёт дополнительный уровень изоляции для сервиса.
05
Step 5
Create workspace
pending ›
Создать директорию, где будет работать агент.
bash
mkdir /home/openclaw mkdir /home/openclaw/workspace cd /home/openclaw
🔒 Security note: Workspace ограничивает область доступа агента — он не может выйти за пределы этой директории.
06
Step 6
Install OpenClaw
pending ›
Скачать и запустить OpenClaw.
bash
git clone https://github.com/openclaw/openclaw cd openclaw

Запустите сервис:

bash
docker compose up
07
Step 7
Configure AI provider
pending ›
Подключить языковую модель (OpenAI или Anthropic).

Добавьте ваш API-ключ в конфигурацию:

env
OPENAI_API_KEY=your_key_here
🔒 Security note: API-ключи нельзя хранить прямо в коде. Используйте .env-файлы и не коммитьте их в репозиторий.
08
Step 8
Create Telegram bot
pending ›
Создать бота через BotFather и получить токен.
  • Откройте Telegram
  • Найдите @BotFather
  • Введите команду:
telegram
/newbot
  • BotFather запросит имя бота
  • Затем username бота
  • После этого вы получите токен
Пример токена
123456:ABCDEF...
09
Step 9
Connect Telegram to OpenClaw
pending ›
Добавить токен бота в конфигурацию OpenClaw.
env
TELEGRAM_BOT_TOKEN=your_token_here

После этого отправьте боту тестовое сообщение:

telegram
hello

Если бот ответил — интеграция работает.
