OpenClaw Setup

Разверните собственного AI-ассистента и подключите его к Telegram.

В этом практическом гайде вы развернёте OpenClaw — систему AI-агента, которая может выполнять команды, работать с сервисами и автоматизировать задачи.

После выполнения инструкции у вас будет:

  • Работающий OpenClaw-инстанс
  • Telegram-интерфейс управления
  • Изолированная инфраструктура
  • Базовая защита системы

~60–90 минут

Лаборатория установки

Следуйте шагам ниже и отмечайте их по мере выполнения.

Прогресс установки 0%

Чеклист

1
Создать сервер
Создайте удалённый сервер для OpenClaw
🧭 Цель

Создать удалённый сервер для OpenClaw.

⚙️ Действия
  1. Перейдите: cloud.digitalocean.com
  2. Нажмите: Create → Droplet
  3. Выберите: Ubuntu 22.04
  4. Выберите тариф: Basic → $6/month
  5. Нажмите: Create Droplet
✅ Проверка результата

Вы получите IP-адрес. Пример:

134.209.xxx.xxx
📸
Страница создания Droplet
🔒 Безопасность

OpenClaw запускается на отдельном сервере, чтобы изолировать систему.

2
Настроить SSH
Подключитесь к серверу безопасным способом
🧭 Цель

Подключиться к серверу безопасным способом.

💻 Команды
ssh-keygen
cat ~/.ssh/id_rsa.pub
ssh root@SERVER_IP
📸
Пример подключения
✅ Проверка результата

Вы увидите приветствие: Welcome to Ubuntu 22.04

🔒 Безопасность

SSH-ключ защищает сервер от атак на пароль.

3
Обновить систему
Установите последние обновления безопасности
🧭 Цель

Обновить системные пакеты до актуальных версий.

💻 Команды
apt update
apt upgrade
✅ Проверка результата

Все пакеты обновлены, нет ошибок в выводе.

🔒 Безопасность

Обновления закрывают уязвимости.

4
Установить зависимости
Git и Docker для работы OpenClaw
🧭 Цель

Установить необходимые инструменты.

💻 Команды
apt install git docker.io
  • 📦 Git — скачивает код
  • 🐳 Docker — запускает OpenClaw
✅ Проверка результата

Команды git --version и docker --version выводят версии.

5
Создать workspace
Рабочая директория для агента
🧭 Цель

Создать изолированную рабочую директорию.

💻 Команды
mkdir /home/openclaw
mkdir /home/openclaw/workspace
✅ Проверка результата
ls /home/openclaw/workspace
🔒 Безопасность

Агент будет работать только внутри workspace.

6
Установить OpenClaw
Скачайте и запустите OpenClaw
🧭 Цель

Клонировать репозиторий и запустить OpenClaw через Docker.

💻 Команды
git clone https://github.com/openclaw/openclaw
cd openclaw
docker compose up
✅ Проверка результата

В терминале появятся логи запуска OpenClaw. Контейнер запущен и работает.

7
Создать Telegram бота
Настройте интерфейс управления через Telegram
🧭 Цель

Создать Telegram-бота для управления OpenClaw.

⚙️ Действия
  1. Откройте Telegram
  2. Найдите: @BotFather
  3. Введите: /newbot
  4. Следуйте инструкциям BotFather
  5. Скопируйте токен бота
📸
Диалог с BotFather
✅ Проверка результата

BotFather пришлёт вам токен вида: 1234567890:ABCdefGhIjKlMnOpQrStUvWxYz

🔒 Безопасность

Никогда не делитесь токеном бота. Храните его в безопасном месте.

Усиление безопасности

По умолчанию OpenClaw может быть уязвим к prompt injection. Добавим несколько уровней защиты.

🏖
Sandbox
Изоляция выполнения
apt install bubblewrap

Bubblewrap запускает команды агента в изолированном namespace.

📂
Ограничение файлов
Контроль доступа

Агент может работать только внутри:

/home/openclaw/workspace
🔐
Vault для секретов
Шифрование данных
apt install sops age

Секреты будут храниться зашифрованными.

🚫
DLP — Data Loss Prevention
Блокировка утечки

Система блокирует отправку:

  • ✕ API ключей
  • ✕ .env файлов
  • ✕ секретов

Troubleshooting

Решения частых проблем

●
Не подключается сервер

Проверьте:

  • → IP адрес
  • → SSH ключ
  • → Порт 22
●
OpenClaw не отвечает

Проверьте статус контейнеров:

docker ps
●
Telegram бот не отвечает

Проверьте токен бота. Убедитесь, что он правильно скопирован в конфигурацию OpenClaw.
